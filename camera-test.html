Troubleshooting Steps:

  1. Click "Test Camera Access" - this should prompt for permission
  2. If no prompt appears, click "Check Permissions" to see current status
  3. If permission is denied, try these steps:
    • Refresh the page and try again
    • Check your browser's camera permissions (look for camera icon in address bar)
    • Try a different browser (Chrome, Firefox, Safari)
    • Make sure no other apps are using your camera
